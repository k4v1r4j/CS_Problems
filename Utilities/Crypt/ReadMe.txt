File Encrypt/Decrypt Tool v1.0
==============================

Principle:
=========
	This command line tool is based on the principle of seed based encr
algorithm given in the problem statement.

How to Run:
===========
./seed_encrypt -t <Encrypt/Decrypt> -i <input> -o <output> -c <config> -s <seed> [-v]

The arguments for -t , -i , -o ,-c and -s is mandatory. whereas for -v is optional.
-v displays the verbose output of the tool
